# FMP_Project

🎓 FMP (Friend Matching Platform)

학교, 학년, 학과 정보를 기반으로 새로운 친구를 매칭하고
채팅할 수 있는 웹 플랫폼입니다.

본 프로젝트는 팀 프로젝트로 시작되었으며,
프로젝트 종료 이후에는 개인적으로 오류 수정 및 기능 개선 작업을 진행하였습니다.
***


🔍 Overview
- 프로젝트 유형: 팀 프로젝트 + 개인 유지·개선
- 개발 목적: 조건 기반 친구 매칭 및 채팅 서비스 구현
- 역할: 백엔드 전반 담당 및 프로젝트 종료 후 개인 개선 작업 진행

___


✨ Key Features
- 🧑‍🤝‍🧑 학교 / 학년 / 학과 기반 친구 매칭
- 💬 매칭 성공 시 채팅방 자동 이동
- 🔐 회원가입 및 로그인, 계정 찾기 기능
- 🧠 세션 기반 사용자 인증
- 🎨 사용자 흐름을 고려한 UI/UX 개선

***

🛠 Tech Stack
Backend
- Java, Spring Boot
- MyBatis
Database
- MySQL
Frontend
- HTML, CSS, JavaScript
Tools
- Git, GitHub
- Spring Tool Suite (STS)

***

👤 My Role
- 백엔드 전반 구현 및 데이터베이스 연동
- 회원가입 / 로그인 / 세션 처리
- 매칭 로직 구현 및 상태 관
- 프로젝트 종료 후 단독 디버깅 및 기능 개선
- UI/UX 개선 작업 일부 담당

***

🚧 Post-Project Improvements (Individual Work)
- 프로젝트 종료 이후, 서비스의 완성도를 높이기 위해 개인적으로 다음 작업을 진행했습니다.
- 매칭 성공 안내 후 채팅방 자동 이동 흐름 개선
- 인증 관련 화면 UI 통일 및 중앙 카드 UI 적용
- 학부 선택 시 학과 자동 필터링 기능 추가
- 아이디 / 비밀번호 찾기 결과 UI 개선
- 에러 메시지 표시 방식 개선 (alert → 폼 내부 카드)
- 버튼 및 링크 인터랙션(hover/active) 추가
- 모바일 환경 UI 미세 조정
- 브라우저 알림 요청 방식 개선 (사용자 선택 기반)

***

⚠️ Troubleshooting
개발 과정에서 발생한 다양한 문제를 직접 분석하고 해결했습니다.
- 회원가입 시 500 에러 (DB 설정 오류)
- 로그인 후 세션 유지 문제
- 매칭 중복 발생 문제
- 이전 매칭 정보가 남아 있던 세션 오류
- Controller → View 데이터 전달 누락 문제
👉 자세한 내용은 포트폴리오 내 Troubleshooting 페이지에 정리되어 있습니다.
